# Rename File

This method can be used to rename a file in the filesystem.

### API

```js
    renameFile(oldPath, newPath) => Promise<string>
```

| param   | type   | description            |
| :------ | :----- | :--------------------- |
| oldPath | string | Path with the old name |
| newPath | string | Path with the new name |
