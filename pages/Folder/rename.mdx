# Rename Folder

This method can be used to rename a folder in the filesystem.

### API

```js
    renameFolder(path) => Promise<void>
```

|param|type|description|
|:--|:--|:--|
|oldPath|string|Path with the old name|
|newPath|string|Path with the new name|
