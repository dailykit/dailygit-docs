# Delete Folder

This method can be used to delete a folder and it's nested folders/files recursively in the filesystem.

### API

```js
    deleteFolder(path) => Promise<void>
```

|param|type|description|
|:--|:--|:--|
|path|string|Path where the folder is stored|
