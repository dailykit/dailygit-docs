# Get search files

This method can be used to search files in the filesystem

### API

```js
   searchFiles(path) => Promise<string>
```

|param|type|description|
|:--|:--|:--|
|path|string|Path where the folder is stored|

### Query
<QueryViewer>
   <QueryTab>
{`query searchFiles($fileName: String!){
  searchFiles(fileName: $fileName)
}
`}
   </QueryTab>
   <VariablesTab>
{`{
   "path": "filename"
}
`}
   </VariablesTab>
</QueryViewer>
