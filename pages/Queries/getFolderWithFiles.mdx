# Get folders with files

This method can be used to get the folders along with files from filesystem

### API

```js
   getFolderWithFiles(path) => Promise<object>
```

|param|type|description|
|:--|:--|:--|
|path|string|Path where the folder is stored|

### Query
<QueryViewer>
   <QueryTab>
{`query {
   getFolderWithFiles($path: String) {
      getFolderWithFiles(path: $path) {
         name
         path
         type
         size
         createdAt
         content
         children {
            name
            path
            type
            size
            createdAt
            content
         }
      }
   }
}
`}
   </QueryTab>
   <VariablesTab>
{`{
   "path": "../apps/appname/data/entityname/file.json"
}
`}
   </VariablesTab>
</QueryViewer>
