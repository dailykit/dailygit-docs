# Get files

This method can be used to get all the files within a folder

### API

```js
   getFiles(path) => Promise<[File]>
```

|param|type|description|
|:--|:--|:--|
|path|string|Path to the folder to get the files inside|

### Query
<QueryViewer>
   <QueryTab>
{`query getFiles($path: String!){
  getFiles(path: $path) {
    type
    size
    createdAt
    commits
    lastSaved
    path
    name
    content
  }
}
`}
   </QueryTab>
