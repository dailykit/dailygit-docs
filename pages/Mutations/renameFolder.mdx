# Rename Folder

This method can be used to rename a folder in filesystem

### API

```js
   renameFolder(oldPath, newPath) => Promise<Result>
```

| param   | type   | description                     |
| :------ | :----- | :------------------------------ |
| oldPath | string | Path where the folder is stored |
| newPath | string | Path with new folder name       |
