# Draft File

This method can be used to save a file as a draft in filesystem

### API

```js
   draftFile(path, content) => Promise<Result>
```

|param|type|description|
|:--|:--|:--|
|path|string|Path where the file is stored|
|content|string|Content to store in the file|
