# Remove and Commit

This method can be used to remove a file from git index and commit it.

### API

```js
    removeAndCommit(path, author, committer) => Promise<sha>
```

|param|type|description|
|:--|:--|:--|
|path|string|Relative path where the file stored|
|author|object|Contains email and name of author|
|committer|object|Contains email and name of committer|
